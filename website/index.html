Stop losing the tab you meant to keep.

Tabguard adds a deliberate close confirmation to the tab you choose. Arm it from the extension popup, then Chrome asks before the protected page closes, reloads, or navigates away.

A simple tab-level guard

Click Tabguard while you are on a page that matters. The extension injects a lightweight listener into that tab only, so other tabs keep behaving normally.

  1. Open a tab. Use any regular website page you want to protect.
  2. Arm Tabguard. Choose Protect this tab from the extension popup.
  3. Confirm exits. Chrome asks before close, reload, or navigation can finish.

Install from GitHub

Download the repository, then load the extension folder in Chrome. Tabguard runs locally from the files you install.

  1. Get the repo. Open the Tabguard GitHub repository and choose Download ZIP, or clone it with Git. If you download the ZIP, extract it before continuing.
  2. Open Chrome extensions. Go to chrome://extensions and turn on Developer mode.
  3. Load the extension. Click Load unpacked and select the extention folder from the repo.

What Chrome controls

The close confirmation is Chrome's native dialog. For security and usability, pages and extensions cannot customize the warning text.

  • Works on normal web pages, not internal pages like chrome://extensions.
  • The page must receive user interaction before Chrome will show the dialog.
  • Protection is per tab and can be disabled from the same popup.
